Form & PHP

Tag Form (<from>)

Tag form merupakan tag yang digunakan saat kita akan membuat sebuah form dalam HTML.

Dalam tag form kita dapat menggunakan atribut name, atribut action, dan atribut method dengan value get dan post. Saat kita menggunakan method get maka kita dapat melihat hasil input kita di browser tetapi jika kita menggunakan method post maka kita tidak dapat melihat hasil input kita di browser. Selain itu juga, kita dapat menggunakan atribut target pada form kita.

Tag Fieldset (<fieldset>)

Tag ini merupakan tag yang berfungsi untuk mengkotakan form yang ada pada halaman website kita.

Tag legend (<legend>)

Tag legend berfungsi untuk memberikan judul pada bagian fieldset form kita.

Tag Input(<input>)

Tag input merupakan tag yang digunakan saat kita mau memasukan input kedalam halaman HTML kita.

Dalam tag input, terdapat atribut type dan value dari atribut type dapat disesuaikan dengan input yang akan dimasukan. Value dari atribut type adalah

  • Text
    Type text merupakan value yang membuat pengguna dapat memasukan input berupa text dalam form di website kita. Jika kita ingin memunculkan input type text lebih dari satu baris (seperti contohnya kolom komentar), kita dapat menggunakan tag textarea.

    • Type Text

    Tag Textarea

  • Password
    Type password merupakan value yang membuat pengguna dapat memasukan input berupa pass dalam sebuah form pada website kita. Saat menggunakan type password, kita hanya melihat input yang kita masukan berubah menjadi simbol.


  • Submit
    Type submit merupakan value yang digunakan oleh pengguna untuk mengirimkan input yang telah dimasukan pada website kita.

  • Reset
    Type reset merupakan value yang digunakan oleh pengguna untuk menghapus semua input yang telah dimasukan pada website kita. Type reset hanya berfungsi untuk input yang dimasukan oleh pengguna saja dan input yang sudah otomatis dimasukan oleh computer tidak akan terhapus.

  • Hidden
    Type hidden merupakan value yang digunakan pada atribut type untuk menampilkan input yang tidak dapat terlihat pada website kita.

  • Number
    Type number merupakan value yang digunakan saat input yang dimasukan hanya berupa angka saja sehingga input huruf ataupun lainnya tidak dapat dimasukan pada type ini. Kita dapat menggunakan atribut min dan max untuk menentukan nilai minimal dan maksimal dari type number. Selain itu juga, kita dapat menggunakan atribut step untuk menentukan interval antara 2 bilangan.

  • Range
    Type range merupakan value yang digunakan untuk membuat pengguna memberikan input menggunakan sebuah glider.

  • File
    Type file merupakan value yang digunakan untuk membuat pengguna memberikan input kepada website dalam bentuk file. Kita dapat menggunakan atribut multiple agar kita dapat memberikan input file lebih dari 1 input.

  • Image
    Type image merupakan value yang digunakan untuk membuat gambar seperti button. Kita dapat menggunakan atribut seperti pada tag img agar gambar dapat muncul dan terlihat sesuai dengan keinginan kita.

  • Color
    Type color merupakan value yang digunakan untuk membuat pengguna memberikan input kepada website dalam bentuk warna sehingga input yang akan muncul adalah warna

  • Time Type local merupakan value yang digunakan untuk membuat pengguna memberikan input kepada website dalam bentuk jam.

  • Datetime-local Type datetime-local merupakan gabungan antara type date dan time. Kita juga dapat menggunakan atribut min dan max untuk menentukan tanggal minimal dan tanggal maksimal yang dapat kita masukan pada input tersebut.

  • Month Type month merupakan value yang digunakan untuk membuat pengguna memberikan input kepada website dalam bentuk bulan.

  • Week Type month merupakan value yang digunakan untuk membuat pengguna memberikan input kepada website dalam bentuk tanggal dan bulan keberapa.

Saat kita akan menggunakan tag input, kita dapat menggunakan beberapa atribut yang dapat memodifikasi input yang akan diterima. Atribut yang dapat kita gunakan adalah

Name : Untuk menuliskan id dari sebuah bagian.
Value : Untuk menuliskan isi dari input text secara otomatis
Readonly : Untuk membuat input dari value tidak dapat dirubah oleh pengguna
Disable : Untuk membuat input pada form tidak dapat diakses oleh pengguna
Maxlength : Untuk membatasi jumlah input yang masuk pada tag input
Required : Untuk membuat sebuah input harus diisi oleh pengguna
Placeholder : Untuk memberikan deskripsi singkat pada kolom input. Saat kita menggunakan atribut ini, kita tidak dapat menggunakan atribut value agar placeholder dapat terlihat.

Dalam penggunaan tag input, terdapat beberapa type yang memungkinkan kita untuk memilih input dari pilihan input yang telah disediakan pada website. Type yang dimaksud adalah :

  • Radiobutton

Type radiobutton merupakan type yang digunakan pada website kita untuk memberikan pilihan input dalam bentuk button lingkaran. Untuk penggunaan type radiobutton, kita hanya dapat memberikan 1 input saja.

Pilihlah makanan yang enak dimakan:

Nasi Goreng Mie Goreng Bihun Goreng

Pada type ini, kita dapat menggunakan atribut name sebagai id dari tag input kita dan atribut value untuk jawaban yang akan dimunculkan pada website. Pada type ini juga, kita harus menggunakan atibut name yang sama agar hanya salah satu pilihan saja yang dapat dipilih.

  • Check Box

Type checkbox merupakan type yang digunakan pada website kita untuk memberikan pilihan input dalam bentuk kotak checklist. Pada type ini, kita dapat memasukan lebih dari 1 input.

Pilihlah makanan yang enak dimakan:

Nasi Goreng Mie Goreng Bihun Goreng

Pada type ini, kita dapat menggunakan atribut name sebagai id dari tag input kita dan atribut value untuk jawaban yang akan dimunculkan pada website. Pada type ini juga, kita harus menggunakan atibut name yang berbeda agar pilihan yang dipilih bisa banyak tetapi jika kita ingin hanya dapat memilih 1pilihan, kita dapat menggunakan value name yang sama. Selain itu juga, kita dapat menggunakan atribut check untuk menchecklist otomatis kotak checkbox yang ada pada website kita.

  • Como box

Type comobox merupakan type yang digunakan pada website kita untuk memberitkan pilihan input dalam bentuk kotak pilihan. Pada type ini, kita hanya dapat memasukan 1 input saja ataupun lebih dari 1 input.

Pilih Makanan yang enak dimakan:

Pada type ini, kita dapat menggunakan tag select lalu menggunakan tag option. Setelah itu, kita dapat menggunakan atribut value dan menuliskan pilihan input yang akan kita masukan ke dalam comobox.

Untuk membuat judul pada comobox kita dapat memasukan tag optgroup lalu atribut label dengan value nama dari judul sehingga saat kita membuka comobox di website kita, kita akan melihat judul copada comobox yang ada pada website kita.

Untuk menampilkan berapa banyak pilihan yang akan ditampilkan pada comobox, kita dapat menggunakan atribut size dengan value sesuai keinginan kita sehingga saat kita meliha comobox, comobox akan menampilkan beberapa pilihan sesuai dengan value yang dimasukan pada size. Selain itu juga, untuk memilih lebih dari 1 input pada comobox, kita dapat menggunakan atribut multiple.

Dalam penggunaan beberapa type diatas, jika kita ingin menampilkan beberapa pilihan pada website kita maka kita harus memasukan manual beberapa tag input dengan type diatas agar pilihan input yang ingin kita munculkan dapat masuk ke dalam website kita.

Tag Textarea

Tag textarea merupakan tag yang berfungsi untuk membuat tempat input lebih dari 1 baris. Untuk membatasi jumlah baris yang bisa diinput, kita dapat menggunakan atribut rows dan juga untuk memperbesar textarea, kita dapat menggunakan atribut cols.
Contoh dari penggunaan tag textarea dapat dilihat diatas.

PHP

PHP(Hypertext Preprocessor) merupakan Bahasa server side scripting yang dapat digunakan oleh kita untuk membuat website kita menjadi lebih responsive dan kinerja yang dilakukan menjadi lebih bagus. Untuk menggunakan php, kita dapat menginstal xampp dahulu.

Xampp merupakan sebuah aplikasi yang berfungsi sebagai server local/localhost yang dapat digunakan untuk membuka file PHP kita. Untuk menginstal xampp, kita dapat menginstalnya melalui google atau mengikuti panduan pada video dibawah

Cara Menginstal XAMPP

Saat kita mau membuat file PHP pada komputer kita, kita dapat masuk kedalam file xampp lalu setelah kita menginstal xampp, kita masuk kedalam folder xampp pada drive kita. Setelah itu cari folder htdocs kemudian masuk kedalamnya. Lalu kita buat sebuah file yang akan menjadi tempat untuk menyimpan file PHP kita. File pada htdocs digunakan untuk membuka/menyimpan file php kita.

Saat kita akan membuat file pPHP pada visual studio code, kita dapat menggunakan struktur seperti HTML tetapi pada bagian body, kita perlu menambahkan <?php kemudian ditutup dengan ?> agar computer mengetahui bahwa isi dari bagian body menggunakan bahasa php.

Untuk menampilkan text pada file PHP, kita dapat menuliskan echo lalu text yang akan ditampilkan ditulis diantara kutip dua (â€œ) lalu diakhiri dengan tanda titik coma(;). Selain itu juga, kita dapat menggunakan kode HTML dalam file PHP. Untuk penulisan kode HTML, kita dapat menuliskannya seperti saat kita menuliskan text pada file PHP.

Selain itu juga, kita dapat menuliskan komentar pada file PHP menggunakan tanda garis miring(/) sebanyak 2 kali atau menggunakan tanda hashtag/pagar(#). Jika kita akan memberikan comment lebih dari 1 baris maka kita dapat menggunakan tanda garis miring dan bintang (/*) kemudian ditutup dengan tanda bintang dan garis miring (*/).

Untuk variable pada file PHP, kita menggunakan tanda dollar ($) kemudian nama variabelnya. Untuk menuliskan value variable, kita dapat menggunakan tanda sama dengan(=) lalu masukan value variable tersebut dan kemudian tutup vairabel menggunakan tanda titik koma(;).

Saat kita ingin menampilkan file PHP kita pada browser, kita dapat mengaktifkan xampp lalu aktifkan apache. Setelah itu, kita dapat mengetikan localhost/(nama file kita) kemudian folder pun akan browser kita.

Pembuatan Form pada PHP

Untuk membuat form pada file php, kita dapat membuatnya seperti dalam pembuatan form di fie HTML. Pada pembuatan form, kita tidak memerlukan tag php () karena pada pembuatan form, kita tidak menggunakan elemen PHP.

Untuk penulisan pembuatan form, kita dapat langsung menggunakan tag form lalu menggunakan atribut nama, atribut action, dan atribut method dengan value get atau post. Setelah itu, kita dapat menggunakan tag input dengan atribut type yang kita inginkan dan atribut name dan value sesuai dengan keinginan kita.

Jika kita ingin hasil input yang telah kita input muncul pada halaman baru, kita dapat menggunakan elemen PHP untuk membuat hal tersebut. Untuk pembuatannya, kita dapat membuka file php halaman yang akan menampilkan hasil input kita kemudian masukan tag PHP lalu buat sebuah variable dengan valuenya yaitu variable GET ($get) kemudian masukan nama dari tag input yang ingin kita tampilkan hasil inputnya. Setelah itu, kita dapat menggunakan echo kemudian menuliskan nama variable yang telah kita buat sehingga saat dijalankan, hasil input yang telah kita masukan akan muncul pada halaman baru.
