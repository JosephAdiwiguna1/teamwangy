    Tentang HTML
HTML (Hypertext Markup Language) adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. HTML terditri dari serangkaian elemen. Elemen-elemen itu lah yang memberitahu kepada browser cara untuk menampilkan konten. HTML dapat dibantu oleh teknologi seperti Cascading Style Sheets (CSS) dan bahasa skrip seperti PHP, JavaScript, Python dan lainnya._HTML ini diibaratkan sebagai tulang (struktur) dari sebuah halaman web, dan CSS sebagai kulitnya atau tampilan.


      Fungsi HTML
  HTML berfungsi untuk
 
  1. Membuat halaman web.
  2. Menampilkan berbagai informasi di dalam sebuah browser Internet.
  3. Membuat link menuju halaman web lain dengan kode tertentu (hypertext).
  4. Menandai sebuah teks pada suatu laman.
  5. Menampilkan tabel, video, gambar, dan lainnya.
  6. Menandai elemen dan membuat form
 

Untuk membuat sebuah file html kita memerlukan sebuah text editor, kira” apa yahh code editor yang akan dipakai ?
Yak betull…. text editor yang akan kita pakai adalah Visual Studio Code.
   

Visual Studio Code adalah adalah perangkat lunak penyunting kode-sumber buatan Microsoft untuk Linux, macOS, dan Windows. Visual Studio Code menyediakan fitur seperti penyorotan sintaksis, penyelesaian kode, kutipan kode, merefaktor kode, pengawakutuan, dan Git. Microsoft merilis sumber kode Visual Studio Code di repositori GitHub dengan lisensi MIT (Code - OSS), sedangkan biner yang dibangun oleh Microsoft tidak dirilis dengan lisensi MIT dan merupakan perangkat lunak berpemilik.
